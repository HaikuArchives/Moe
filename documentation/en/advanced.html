Advanced Settings

Lock Position

Click mascot with seconday mouse button, select "Lock" from menu, mascot position will be locked. Locked mascot stay on current position even if it is dragged.

Click mascot with holding Alt+Shift+Ctrl, two item "LockX" and "LockY" appears in advanced menu. Select "LockY", cancel Y directional lock, you can also drag mascot through up and down.

Change Size

Click mascot with seconday mouse button, select "Size", you can change mascot size to prefered proportion. When mascot is too big against screen resolution, adjust with this function.

Set Update Frequency

Click mascot with seconday mouse button, select "Wink", you can set wink frequency. When you feel mascot's wnik frequency is too few or too many, adjust with this function.

Click mascot with seconday mouse button, select "Polling", you can set polling interval to detect active window. When CPU load is too high, adjust with this function.

Click mascot with seconday mouse button, select "Redraw", you can set delay till mascot's redraw. When garbage is drawn behind mascot, adjust with this function.

Ignore Particular Application

Click mascot with seconday mouse button, select "Ignore This App", mascot will ignore current application after this.

When you want to restore an application from ignore list, click mascot with holding Alt+Shift+Ctrl, select an application from "Avoid from Ignore List" sub menu.

When you want to show mascot only on your prefered application or window, click mascot with holding Alt+Shift+Ctrl, select "This Application" or "This Window of This Application" from "Show Only on"
